# CUBi NOI Calculator v2.0

**Community Utility Billing Integration — Net Operating Income Calculator**

## 🎯 What It Does
Single-file browser tool for GreenSmart's office-to-storage conversion model. Calculates NOI before/after conversion with 10 revenue streams, zero-capital financing, and sensitivity analysis.

## 🚀 Live Demo
Open `CUBi_NOI_Calculator_v2.html` in any browser. Works offline. No server needed.

## 💰 Key Features
- 10-stream CUBi revenue model ($546,804/year per building)
- $1 Well Revolution zero-capital financing
- Sensitivity analysis (cap rate, occupancy, interest, vacancy)
- PDF + CSV export
- Dark mode, mobile field mode, client view
